## 🎯 Как пользоваться приложением

### 1. **Home экран**
   - Просмотр новостей города
   - Поиск по названию или содержимому
   - Сохранение избранных новостей
   - Клик для полного чтения

### 2. **Alerts экран**
   - Просмотр всех оповещений
   - Фильтрация по категориям
   - Фильтрация по серьезности
   - Поиск оповещений

### 3. **Map экран** (существующий)
   - Просмотр мест на карте
   - Переход к деталям места

### 4. **Profile → Settings**
   - Редактирование профиля
   - Установка пароля приложения
   - Настройка типов и каналов уведомлений
   - Выбор языка
   - Просмотр активных сессий
